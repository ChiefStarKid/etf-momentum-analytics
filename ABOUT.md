# About

This project was built by Joseph Solomon.

I'm a project manager and operations lead by trade, but I've spent the past year building systematic analytics tools — signal pipelines, backtesting frameworks, and data automation — mostly because the problems were interesting and the existing tools didn't do what I needed.

This repository is one output of that work: a structured attempt to measure whether a weekly momentum signal source carries genuine directional edge, and to figure out under what conditions that edge is strongest.

---

## What problems I work on

- Signal design and backtest methodology for momentum and regime-based strategies
- Building analytics pipelines from raw data sources to structured results
- Automating recurring data workflows (extraction, transformation, scoring, reporting)
- Translating quantitative findings into decision-relevant outputs
