# Contributing

Thanks for cloning. Here's how to engage:

## Star first

If the methodology is useful — even as a reference — a star helps other quants find it.

## Found an issue with the analysis?

Open an issue describing:
- Which backtest version or finding you're questioning
- What data or reasoning you're working from

Methodological challenges are welcome. This is research, not a finished product.

## Want to extend it?

The most useful extensions would be:
- Comparing these signals against independent exhaustion measures (RSI divergence, volume climax)
- Testing on additional ETF universes or timeframes
- Validating on out-of-sample data

Open an issue before coding — I'd rather co-design the extension than review a closed PR.

## Questions

Use GitHub Discussions.
